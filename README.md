A simple chart library for C++

# Air Chart 1.0.0

# Installation:

## Build for Linux
```
git clone https://github.com/Bot-Kerem/Air-Chart.git
cd Air-Chart
./build_linux.sh
```

## Build for Windows
```
Currently not supported
```

## Build for Mac
```
Currently not supported
```

## CMake

```

find_package(AirChart REQUIRED)

target_link_libraries(PROJECT_NAME AirChart)

```
